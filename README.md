# Bland–Altman解析ツール

2つの測定法・検査法・評価法の**一致性（agreement）**を、ブラウザ上で簡単に解析できるHTMLツールです。  

平均差（bias）、95%一致限界（Limits of Agreement）、Bland–Altman plot を自動作成し、結果の解釈コメントまで表示します。

インストール不要・無料・オフライン使用可能です。

---

## 主な用途

- オートレフ値 vs 自覚屈折値
- 新型機器 vs 従来機器
- 検者A vs 検者B
- 術前予測値 vs 術後実測値
- 2種類の眼科検査機器の比較
- 研究データの初期確認

---

## 主な機能

### 解析機能

- 症例数（n）
- 平均差（bias）
- 差の標準偏差（SD）
- 95%一致限界（LoA）

### 作図機能

- Bland–Altman plot 自動生成
- biasライン表示
- 上下LoAライン表示
- 外れ値の強調表示

### サポート機能

- Excelから2列データをそのまま貼り付け可能
- PNG画像保存
- 結果の自動解釈コメント表示

---

## 使用方法

### 1. データ準備

Excelで2列のデータを作成してください。

| A列 | B列 |
|----|----|
| 測定法A | 測定法B |

例：

| A | B |
|---|---|
| 0.25 | 0.00 |
| 0.50 | 0.25 |
| -0.25 | -0.50 |

### 2. 貼り付け

2列をコピーして、HTML画面の入力欄へ貼り付けます。

### 3. 解析

「解析する」を押すだけです。

---

## 結果の見方

### bias（平均差）

- 正の値：AがBより高めに出る傾向
- 負の値：AがBより低めに出る傾向

### 95%一致限界（LoA）

個々の測定値の差が、どの範囲に収まりやすいかを示します。

### 重要ポイント

統計学的有意差よりも、

**その差が臨床的に許容できるか**

で判断することが重要です。

---

## 注意事項

このツールは基本版です。

以下には対応していません。

- 左右眼のクラスタ補正
- 繰り返し測定データ解析
- 比例誤差の回帰型Bland–Altman解析
- 正規性検定
- 多群比較

必要に応じてR / Python / SPSS等で追加解析してください。
